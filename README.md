# Quiz Game Application

## 📌 Project Overview

The Quiz Game Application is a simple and interactive web-based quiz built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions, navigate through the quiz, and view their final score at the end.

## 🚀 Features

* Multiple-choice quiz questions
* Interactive user interface
* Instant answer selection
* Score calculation and display
* Restart quiz functionality
* Responsive design for different screen sizes

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

Quiz-Game-App/
│
├── index.html
├── style.css
└── script.js

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Open `index.html`.
5. Right-click and select **Open with Live Server**.
6. The application will open in your default browser.

## 📸 Application Workflow

1. User starts the quiz.
2. A question with four options is displayed.
3. User selects an answer.
4. User moves to the next question.
5. Final score is displayed after completing all questions.
6. User can restart the quiz and play again.
